# Machine learning  2020-2021 - UIB 

Aquesta és la meva proposta de pràctica de Machine Learning. 

## Instal·lació

En primer lloc clonam aquest repositori:
```
     https://github.com/EnergyWoxt/Pr-ctica-Machine-Learning-IA
```

**Conda**

Executam el següent comandament amb conda per instal·lar l'entorn per a executar la pràctica:
```
    conda env create --file entorn.yml
```

**Jupyter Notebook**


Seleccionam el document *Practica.ipynb* per executar la pràctica.
